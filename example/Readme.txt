This Kinect library is a simple OpenNI and NITE wrapper for Javascript.
This library allows user to use Kinect sensor as input in Internet Explorer when connecting a Kinect sensor, otherwise it can use mouse as input.


How to install?
1. download hiKinect.zip and unzip 
2. Open Terminal and install file install.sh (sudo bash install.sh)
3. This library works well in Chome
4. Install Kinect folder in Chrome extension
